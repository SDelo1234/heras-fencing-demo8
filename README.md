# Heras Fencing UI (Vite + React + Tailwind)

### How to use

1. Replace `src/Heras-fence-ui.jsx` with your full component.
2. Push this folder to a new GitHub repo.
3. Deploy on Vercel:
   - Framework: **Vite**
   - Build: `npm run build`
   - Output: `dist`
   - Install: `npm install`
   - Node.js: 18 or 20

Local dev:

```bash
npm install
npm run dev
```
